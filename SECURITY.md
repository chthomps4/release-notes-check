# Security Policy

## Supported Versions

The latest tagged release is supported for security fixes.

## Reporting a Vulnerability

If you discover a security issue, please do not open a public issue.

Instead, report it privately with:
- a description of the issue
- steps to reproduce it
- affected version
- possible impact

For now, contact the maintainer through GitHub.

## Scope

This project is a GitHub Action for validating release note metadata on pull requests.
Please include any relevant workflow logs or repository setup details when reporting issues.
